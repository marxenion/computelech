Set Operations and Venn Diagrams

Introduction

Sets are treated as mathematical objects. Similarly to numbers, we can perform certain mathematical operations on sets. Below we consider the principal operations involving the intersection, union, difference, symmetric difference, and the complement of sets. To visualize set operations, we will use Venn diagrams. In a Venn diagram, a rectangle shows the universal set, and all other sets are usually represented by circles within the rectangle. The shaded region represents the result of the operation.

Intersection of Sets

The intersection of two sets, A and B, is the set of elements which are in both sets A and B. The intersection of the two sets is written as A ∩ B.

Two sets are called disjoint if they have no elements in common.

  • Example:

    • A = { a, b, c }, B = {k, l, m}

These two sets are disjoint as they have no common elements. Their intersection is the empty set. The intersection of these sets is A ∩ B = { a, b, c } ∩ { k, l, m} = ∅

  • Example:

    • C = { 1, 2, 3, 4 }, D = {2, 4, 6, 7}

The intersection of these sets is C ∩ D = { 1, 2, 3, 4 } ∩ { 2, 4, 6, 7} = { 2, 4 }

Union of Sets

The union of two sets A and B is defined as the set of elements which are either in set A or set B or in both A and B. This operation is denoted by the symbol.

  • Example:

    • A = { a, b, c }, B = {k, l, m}

    The union of the two sets is given by A ∪ B = { a, b, c } ∪ { k, l, m } = { a, b, c , k, l, m }.

  • Example:

    • C = { 1, 2, 3, 4 }, D = {2, 4, 6, 7}

The union of the two sets is given by C ∪ D = { 1, 2, 3, 4 } ∪ { 2, 4, 6, 7 } = { 1, 2, 3 , 4, 6, 7 }.

Principle of Inclusion-Exclusion

The cardinality of a finite set A, denoted by |A| is equal to the number of elements in it. The cardinality of the union of two finite sets A and B is given by the following relationship:

|A ∪ B| = |A| + |B| - |A ∩ B|,

where |A ∩ B| is the cardinality of the intersection of A and B.

he similar formula exists for the union of 3 finite sets:

|A ∪ B ∪ C| = |A| + |B| + |C| - |A ∩ B| - |A ∩ C| - |B ∩ C| + |A ∩ B ∩ C|

Difference of Two Sets

The difference of two sets A and B is the set that contains exactly all elements in A but not in B. The difference of two sets A and B is denoted by A\B or A - B.

  • Example:

    • A = { a, b, c }, B = {k, l, m}

    The difference between two disjoint sets is equal to the initial set. So, we have A \ B = A \ ( A ∩ B ) = A \ ∅ = A = { a, b, c }

  • Example:

    • C = { 1, 2, 3, 4 }, D = {2, 4, 6, 7}

    The difference of two sets C and D is given by C \ D = { 1, 2, 3, 4 } - { 2, 4, 6, 7 } = { 1, 3 }

Symmetric Difference

The symmetric difference of two sets A and B is the set of all elements which belong to exactly one of the two original sets. This operation is written as A △ B or A ⊕ B.

In terms of unions and intersections, the symmetric difference of two sets A and B can be expressed as

A △ B = ( A ∪ B ) \ (A ∩ B).

  • Example:

    • A = { a, b, c }, B = {k, l, m}

    The symmetric difference of two disjoint sets is equal to their union. So, we have A △ B = ( A ∪ B ) \ ( A ∩ B ) = ( A ∪ B ) \ ∅ = A ∪ B = { a, b, c, k, l, m }

  • Example:

    • C = { 1, 2, 3, 4 }, D = {2, 4, 6, 7}

    The symmetric difference of the sets C and D is given by C and D is given by C △ D = ( C ∪ D ) \ ( C ∩ D ) = { 1, 2, 3, 4, 6, 7 } - { 2, 4 } = { 1, 3, 6, 7 }

Complement of a Set

The complement of a set A is the set of elements in the given universal set U that are not elements of A. The complement of A is denoted by Ac or sometimes A', or sometimes A¯. So by definition, we have

Ac = U\A

  • Example:

    • Let the universal set be U = { a, b, c, d, e, f}. If A = { a, b, d, f} then the complement of A is given by

    Ac = U\A = { c, e }

  • Example:

    • C = { 1, 2, 3, 4 }, D = {2, 4, 6, 7}

    Suppose the universal set is U = { x ∈ Z ∣ x2 < 20 } and the set A is given by A = { x ∈ Z ∣ − 3 ≤ x < 3 } . Find the complement of A.

U = { − 4 , − 3 , − 2 , − 1 , 0 , 1 , 2 , 3 , 4 } , A = { − 3 , − 2 , − 1 , 0 , 1 , 2 } , ⇒ Ac = { − 4 , 3 , 4 } .
